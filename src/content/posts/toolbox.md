这个博客的第一版工具箱很小：

- Astro 负责站点生成。
- Markdown 负责写作。
- GitHub 负责版本管理。
- Vercel 或 Cloudflare Pages 负责部署。
- RSS 负责订阅。

等文章数量变多以后，可以再加入全文搜索、评论系统和访问统计。

轻量不是功能少，而是每个工具都能解释为什么存在。
